# Family Medicine Knowledge Base — Index

> 個人家庭醫學知識庫。由 LLM 讀取 `raw/` 入面嘅原始資料,編譯成呢個 wiki。

## How to Use
1. 收料擺入 `raw/`(照下面嘅 subfolder 分類)
2. 叫 LLM 讀 `raw/` 然後寫/更新 `wiki/concepts/` 嘅文章
3. 查詢答案 save 入 `wiki/qa/`
4. 每星期 lint 一次,搵矛盾同空白

## Folder Map
- `raw/articles/` — Web Clipper 剪落嚟嘅網上文章
- `raw/papers/` — arXiv / PubMed PDFs
- `raw/textbooks/` — 教科書 chapter(一 chapter 一個 file)
- `raw/guidelines/` — 臨床指引(NICE, AAFP, USPSTF, ESC 等)
- `raw/notes/` — 自己嘅臨床筆記、case log
- `wiki/concepts/` — 編譯後嘅概念文章(主要產物)
- `wiki/summaries/` — 每本書 / 每篇 paper 嘅摘要
- `wiki/qa/` — Q&A agent 產生嘅答案
- `outputs/slides/` — Marp slide decks
- `outputs/charts/` — matplotlib / 視覺化圖表
